SDK Installation

1. Click on “Step 3. Install SDK Components” and you will be prompted with the ACR120U Contactless Reader SDK.

2. Please wait while InstallShield prepares the setup.












 

 

3. Click “Next>” to continue installation.

4. When prompted to choose the destination location, click on “Next>”.

The default path where the SDK will be installed is x:\ Program Files\Advanced Card Systems Ltd\ACR120U Contactless Reader SDK, where x is the drive letter of your local hard drive.

Note: If you want to change the destination path, click on "Browse…." When the Choose Folder window appears, change the path to your preferred folder then, click OK.


5. In the Setup Type window, leave the Typical option selected then, click “Next>”.

6. When prompted with the Select Program Folder window, click “Next>”. Default program folder that will be created is ACR120U Contactless Reader SDK. If you want a different folder, type a new program folder name.

7. Please wait while the software is being configured.

8. ACR120U Contactless Reader SDK installation has been successfully installed. Click “Finish” to complete setup
