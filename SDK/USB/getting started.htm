System Requirements

This kit will require the following system specifications:

- IBM compatible Personal Computer with Intel Pentium 800 MHz Processor or higher;
- Microsoft Windows 98SE / ME / 2000 / XP;
- CD-ROM Drive;
- One available USB port.

SDK Checklist

The ACR120 Contactless Reader SDK contains the following:

- 1 unit ACR120U Contactless Reader;
- 1 unit ACR38DT DualKey;
- 1 piece Combi Card;
- 5 pieces Mifare RF Cards;
- SDK CDROM;
- Quick Guide Manual.

A. Connection Diagram

Note: You can connect the ACR120U Contactless Reader and ACR38DT DualKey devices to your PC anytime AFTER the drivers has been loaded.  

To connect the ACR120U Device to your PC, plug in the USB connector (Type A) into available USB Port.

                     

To connect your ACR38DT to your PC, plug in the USB end of the device into the USB Port of your computer.

                     

Install SDK

Insert the ACR120 SDK CDROM into your CDROM drive. If the screen below does not appear, run x:\ACR120SDK.EXE where x is the drive letter of your CDROM.

Install SDK Components

The default path where the SDK will be installed is x:\Program Files\Advanced Card Systems Ltd\ACR120U Contactless Reader SDK where x is the drive letter of your local Windows drive. Click here to view the SDK Installation Guide.

Click on Step 3. Install SDK Components button from the ACR120U SDK Setup GUI or this link to install the ACR120U SDK components. It includes the following:

o Sample Application – A demo program that uses the ACR120 for various applications.

o Multi-Application Demo Program - This demo program shows how the ACR120 is used in real-world applications. The card is used as stored-value:

- Parking Card;
- Train Ticket Card;
- For Vending Machine;
- For Weighing Machine;
- For Loyalty Application

o Sample Codes - The sample codes are written in 3 programming languages: Delphi (Borland Delphi 7.0); C (Microsoft Visual C++ 6.0); and Visual Basic (Microsoft Visual Basic 6.0). It includes the following:

o ACR120 Antenna Power and RW User Port Sample – shows how to disable the ACR120 reader’s antenna and how to get/set the state of the ACR120 user port (PIN 14). This will cause optimization of the reader’s power consumption.

o ACR120 Copy Block Sample – demonstrates how to copy blocks of bytes from one card region to another.

o ACR120 MultiTag Select Sample – demonstrates how to single out a Mifare card from multiple cards within the ACR120’s range.

o ACR120 RW Block Sample – demonstrates how to read/write blocks of data from/to the Mifare card.

o ACR120 RW EEPROM - demonstrates how to read/write data from/to the ACR120 EEPROM.

o ACR120 RW Inc Dec Value Sample - demonstrates how to increment/decrement value data in the Mifare card.

o ACR120 Status Sample – demonstrates how to get the status of the ACR120U device.

o ACR120 TxDataTelegram Sample - demonstrates how to transfer user specific data frames.

o ACR120 Write Master Key - demonstrates how to write master keys to the ACR120 EEPROM.

o Tools and Utilities

o ACR120 Tool - This program helps user perform contactless card operations using ACR120 and Mifare card(s). You can directly send low-level commands to your ACR120.

o QuickView - This is a utility program that checks if you have properly installed your ACR3x readers.

o User Manuals and Reference Materials - These files are in PDF. You can install the Acrobat Reader from the ACR120U SDK Setup GUI or just click this and run it.

o ACR120U API – specifications on how to program the ACR120 reader using ACS Proprietary driver’s API. Click this to read.

o ACR38DT Technical Specifications – physical and technical specifications for the ACR38DT. Read it.

o ACR120 Technical Specifications – physical and technical specifications for the ACR120. Read it.

o Mifare 1K Card Reference Manual – specifications for the 1K Mifare Contactless Card. Read it.

o Mifare 4K Card Reference Manual – specifications for the 4K Mifare Contactless Card. Read it.

o ACR120S - ACR120U Porting Guide – Specifications on how to port application from ACR120S to ACR120U and vice-versa . Read it.

o ACR120 Wall Mounting Procedures – specifications on how to mount the ACR120 reader . Read it.

o Product and Service Guide - Information on ACS Products and its services. Read it.
